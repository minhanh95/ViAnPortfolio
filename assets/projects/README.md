# Cai dat anh cover / Project Cover Setup

Dat moi anh cover voi ten `cover.jpg` trong dung thu muc du an:

Place each cover image as `cover.jpg` in the corresponding project folder:

- `assets/projects/no-one-magazine/cover.jpg`
- `assets/projects/lofficiel-beauty-award-2025/cover.jpg`
- `assets/projects/vinamilk-green-farm/cover.jpg`
- `assets/projects/one-plus/cover.jpg`
- `assets/projects/x3d-robot/cover.gif`

Ban co the dung `.jpg`, `.jpeg`, `.png`, hoac `.gif`, sau do cap nhat `coverPath` trong `script.js` neu can.

You can use `.jpg`, `.jpeg`, `.png`, or `.gif`, then update each `coverPath` in `script.js` if needed.
